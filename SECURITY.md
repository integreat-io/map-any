# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.3.x   | :white_check_mark: |

## Reporting a Vulnerability

To report a vulnerability, please
[create an issue](https://github.com/integreat-io/map-any/issues) in this
repo.
